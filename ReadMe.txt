# Hotel Booking Web Application

This project is a hotel booking web application that allows users to search for hotels, view details, and make bookings.

## Overview

The application provides a user-friendly interface for searching and booking hotels. Users can log in using email/password or Google authentication, and registered users receive special discounts on flagged hotels.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository: `git clone https://github.com/burak-ince/hotel-booking-app.git`
2. Navigate to the project directory: `cd hotel-booking-app`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the application: `python run flask_app.py`
